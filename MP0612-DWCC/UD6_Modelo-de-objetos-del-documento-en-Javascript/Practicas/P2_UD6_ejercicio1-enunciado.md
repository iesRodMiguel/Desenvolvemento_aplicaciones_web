- En el input type text Nombre --> El usuario debe introducir el nombre en MINUSCULAS. [Esto lo puedo dejar para el final]
- Al pulsar el botón Enviar:
    * Se muestra el nombre del usuario con el formato, tipografia y estilos elegidos al final del documento (después del botón 'Enviar'en el <span id="resultado"></span>)
    * Al mismo tiempo, se abre una ventana donde muestra el nombre del usuario con formato h1, nombre del usuario con formato y efectos elegidos en el formulario y un botón Cerrar.
- Al pulsar el botón Cerrar (en la nueva pestaña), ésta se cierra.
